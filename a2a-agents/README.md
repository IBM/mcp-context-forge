# A2A Agents

Sample A2A agents used for ContextForge integration, compose, Langfuse, and
load testing.

## Agents

### Go

| Agent | Description |
| ----- | ----------- |
| `a2a-echo-agent` | Lightweight A2A echo agent used by the testing profile. |

### Rust

| Agent | Description |
| ----- | ----------- |
| `a2a-echo-agent` | Rust A2A echo agent counterpart with JSON-RPC, agent card, health, and task lookup endpoints. |
