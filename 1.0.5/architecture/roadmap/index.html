    Release 1.0.0-RC3¶

    Release 1.0.0-RC3 - Complete (100%)

    Due: 14 Apr 2026 | Status: Closed Release Candidate 3 - Auth Hardening, Plugin Multi-Tenancy, Rust Runtime & Multi-Arch

MySQL/MariaDB/MongoDB support removed. See CHANGELOG.md for full details.
